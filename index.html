This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Inside my-div
class

classes can be applied to more than one item on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:orange;
}

compound

A conmpund selector allows us to use multuple selectors separated by commas. This allows us to apply the same rules to several selectors.

Here's how the CSS is applied to the element:

div#new-div, li.new-list{
    background-color:blue;
}

child

A child selector allows us to access the nest immediate elements inside the current element.

Here's how the CSS is applied to the element:

ul.my-kids>li{
    background-color:red;
}

Here's what it looks like:

  • I'm in an li inside a ul with a class of "my-kids"

descendant

Using a descendant selector we can access all elements at any depthwitin the current element.

Here's how the CSS is applied to the element:

div.grandkids li{
    background-color:green;
}

Here's what it looks like:

  • I'm in an li inside a div with a class of "grandkids"

multiple classes

We can apply multiple classes to an element by placing class name separated by spaces in the class attribute.

Here's how we hook multiple classes to an element:

class="add-border and add-background"

Here's how the CSS is applied to the element:

.add-border{
    border: 1px solid #000;}
.add-border{
    border: 1px solid #000;}
